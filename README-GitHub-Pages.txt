# Gizem Göçmen — Veri Analizi & Excel/İstatistik Danışmanlığı

Bu klasördeki `index.html` dosyasını **GitHub Pages** ile ücretsiz yayına alabilirsiniz.

## 1) GitHub'da depo (repo) oluştur
- GitHub hesabınıza girin (yoksa ücretsiz açın).
- Sağ üstten **New repository** deyin.
- İsim: `gizemgocmen-site` (farklı da olabilir), **Public** seçin.
- **Create repository**.

## 2) Dosyaları yükleyin
- Bu klasördeki *index.html* ve `assets` klasörünü GitHub deposuna sürükleyip bırakın.
- **Commit changes** butonuna tıklayın.

## 3) Pages'i açın
- Depoda **Settings → Pages** bölümüne girin.
- **Build and deployment** altında: Source: **Deploy from branch**.
- Branch: `main`, Folder: `/root`, **Save**.

## 4) Yayın adresi
- Birkaç dakika içinde sayfanın alt kısmında sitenizin adresi çıkar: `https://<kullanici_adiniz>.github.io/<repo_adi>/`
- Örnek: `https://gizemgocmen.github.io/gizemgocmen-site/`

## 5) Özel alan adı (opsiyonel)
- Bir domain aldıysanız **Settings → Pages** altında **Custom domain** alanına yazıp DNS ayarlarını yapın.

## Notlar
- Sosyal paylaşımlarda düzgün önizleme için `assets/og-image.png` kullanılır (Open Graph).
- Logonuz `assets/logo.png` içindedir.
- Görsel promosyonlar: `assets/promo1.png`, `assets/promo2.png`, `assets/promo3.png`.
